# F1-RL

## Installation instructions

Clone repo:
```bash
git clone https://github.com/abhaybd/F1-RL.git
cd F1-RL
```

Create environment

```bash
conda env create -f env.yaml
conda activate f1-rl
```

Now install f1tenth gym

```bash
git clone https://github.com/f1tenth/f1tenth_gym.git
cd f1tenth_gym
pip install -e .
```
